# @svrx/eslint-config

[![npm](https://img.shields.io/npm/v/@svrx/eslint-config.svg?style=flat-square)](https://www.npmjs.com/package/@svrx/eslint-config)

ESLint config for svrx.


### Usage

```bash
npm i eslint @svrx/eslint-config -D
```

In your `.eslintrc`:

```json
{
  "extends": "@svrx"
}
```
